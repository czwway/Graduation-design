# weixin

> 毕业设计——前端vue+vue-router+iview+axios，后端Express+mongodb

分为前端和后端

后端文件在**back-end**文件夹里

前端端口：8080
后端端口：8081


----------


# 初始化，安装前端依赖
npm install

# 前端环境启动
npm run dev


----------


# 后端服务器开启
-------------**确保开启mongodb**

-------------**cd 进入back-end文件夹**

**初始化，安装后端依赖**

npm install

**开启服务**

npm app
